## Data Engineering Bootcamp — KickOff

### What is Data Engineering?

Data engineering is the practice of designing and building systems for collecting, storing, and analyzing data at scale.

### Key Topics Covered

1. **Data Modelling** — Dimensional modelling, fact tables, slowly changing dimensions
2. **ETL Pipelines** — Extract, Transform, Load workflows
3. **Data Warehousing** — Star schema, snowflake schema
4. **Batch vs Stream Processing** — When to use each approach
5. **Data Quality** — Ensuring accuracy, completeness, and consistency

### Tools & Technologies

- Apache Spark
- Apache Airflow
- dbt (data build tool)
- PostgreSQL / Redshift / BigQuery
